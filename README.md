# Embeddmo Event Manager – Example application

## Overview

This repository contains an example embedded application demonstrating how to use the **Embeddmo Event Manager** library in a structured, event-driven system.

The purpose of this project is to illustrate how software modules can communicate through a centralized event mechanism, enabling clean architecture, loose coupling, and predictable system behavior.

---

## Important notice

The **Embeddmo Event Manager** library is **proprietary (private)** and is **NOT included** in this repository.

As a result:

* The project **will NOT build successfully** out of the box
* Some source files and dependencies are intentionally omitted
* This repository is intended for **demonstration and evaluation purposes only**

## What this example demonstrates

* Event registration and subscription mechanisms
* Event publishing and handling
* Decoupled communication between modules
* Centralized event dispatching
* Deterministic event processing flow
